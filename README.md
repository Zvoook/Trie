# Вариант 1: Trie (Префиксное дерево)
Реализация структуры данных Trie без использования готовых коллекций.

## Основные операции
- insert(String word) — добавление слова в дерево
- contains(String word) — проверка наличия слова
- startsWith(String prefix) — проверка существования слов с префиксом
- getByPrefix(String prefix) — получение всех слов по префиксу
- clear() — очистка дерева

## Реализация
Каждый узел дерева содержит:
- Массив из 26 элементов (по числу букв английского алфавита)
- Флаг isEnd — отмечает конец слова

## Команды в консольном интерфейсе
- `add` — добавить слово
- `print` — показать все слова
- `find` — найти слова по префиксу
- `contains` — проверить наличие слова
- `starts with` — проверить наличие слова
- `clear` — очистить дерево
- `help` — показать справку
- `#` — выход

## Особенности
- Работает только с английскими буквами (a-z)
- Регистр не имеет значения (все приводится к нижнему)
- Нельзя добавить пустую строку или null
- Обработка ошибок с понятными сообщениями

## Сложность операций
- insert O(n)
- containsO(n)
- startsWithO(n)
- getByPrefixO(n + m)
где n — длина слова, m — количество найденных слов
